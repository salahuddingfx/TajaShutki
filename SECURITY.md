# 🛡️ Security Policy

## Supported Versions
Only the latest version of the Multi-Store system is currently supported for security updates.

| Version | Supported |
| ------- | --------- |
| v1.0.x  | ✅         |

## Reporting a Vulnerability
If you discover a security vulnerability within this project, please report it immediately. We take security seriously and will respond as quickly as possible.

You can report vulnerabilities by:
- Creating a private issue on GitHub.
- Contacting the developer directly at **salahuddingfx@gmail.com**.

Please do not disclose the vulnerability publicly until it has been resolved.
